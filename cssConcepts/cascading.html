CSS

O css é lido de cima para baixo. Além disso, são levados em consideração 3 fatores:

  1. Origem do estilo
  2. Especificidade
  3. Importância

inline >> tag style >> tag link

Especificidade

É um cálculo matemático, onde cada tipo de seletor e origem do estilo possuem valores a serem considerados.
Quanto maior o número (ou a soma dos números), mais específico é e, consequentemente, terá o valor aplicado.
Caso tenha o mesmo valor, o que foi citado por último é o que determinará o valor a ser aplicado.

0. Universal selector, combinators e negation pseudo-class (:not())
1. Element type selector e pseudo-elements (::before, ::after)
10. Classes e attribute selectors ([type="radio"])
100. ID selector
1000. Inline

Exemplos:

Esse texto deve ficar na cor vermelha
Esse texto deve ficar na cor verde
Esse texto deve ficar na cor azul
Esse texto deve ficar na cor verde, EXCETO a palavra texto,
pois a palavra texto tem o valor 12 (10 da classe mais 2 dos selectors (div e strong))
Esse texto deve ficar na cor verde, inclusive a palavra texto
Esse texto deve ficar na cor rosa, inclusive a palavra texto
Esse texto deve ficar na cor verde

!Important

Não é considerada uma boa prática usar o !important, mas nem sempre é possível evitar o uso.
Um dos casos em que costuma ser "permitido" usar é quando está sendo utilizada alguma biblioteca externa onde não se tem o controle.
